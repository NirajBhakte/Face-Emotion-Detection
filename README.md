🎭 Face Emotion Detection using Deep Learning

📌 Description

A real-time Face Emotion Detection system built using Deep Learning and Computer Vision. The model uses a Convolutional Neural Network (CNN) to classify facial expressions into emotions such as Angry, Happy, Sad, Surprise, and Neutral from live webcam input.

🧠 Technologies Used
Python
TensorFlow / Keras
OpenCV
NumPy
Matplotlib
📂 Dataset

FER-2013 Facial Expression Dataset (Kaggle)

⚙️ Features
🎥 Real-time emotion detection using webcam
🧠 CNN-based classification model
😊 Multi-class emotion recognition
⚡ Fast and efficient processing
🚀 How to Run
Install dependencies
pip install -r requirements.txt
Train the model
python src/train.py
Run live detection
python src/live_detection.py
📊 Results
Validation Accuracy: ~67%
Works effectively under normal lighting conditions
🧠 Conclusion

This project demonstrates the practical application of deep learning for understanding human emotions through facial expressions in real time.
